# Cadastro de Veículos em Java

Projeto acadêmico em Java seguindo a estrutura com dois arquivos (`Input.java` e arquivo principal), menu com `switch` e repetição com `do-while`.

## Funcionalidades implementadas

- Cadastro de veículo
- Validação de duplicatas ignorando maiúsculas e minúsculas
- Listagem de veículos com ordenação alfabética manual
- Exibição do total de veículos cadastrados
- Busca por nome ignorando maiúsculas e minúsculas
- Mensagem para lista vazia nas operações de listar e buscar
- Edição de veículo por índice com validações
- Remoção por índice
- Remoção por nome
